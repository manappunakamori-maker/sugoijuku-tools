想定モード
楽観：効率よく進められる場合の最短見積もり（下限値）
標準：平均的なペースの見積もり（中間値）
厳しめ：定着に時間がかかる場合の最長見積もり（上限値）
Phase 01   未学習 → Lv1（入試基礎）
Phase 02   Lv1 → Lv2（MARCH・関関同立）
Phase 03   Lv2 → Lv3（早慶上理）
累計   0 → 各レベル到達までの合計時間
※ 算出の前提： 各数値は1シーズン=90日を単位とし、1日の学習時間は科目単体の時間で計算しています。 複数科目を並行学習する場合は1日あたりの総学習時間を各科目に分配してください。 漢文はLv2が上限。数学は文系・理系で別集計。理科は1科目あたりの時間です（2科目選択の場合は×2）。

※ 時間幅について： 楽観・厳しめの差は個人の学習効率・定着度・使用教材・指導有無などによるばらつきを反映しています。
