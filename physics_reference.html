G-01 変位・速度・加速度の定義（運動の記述） 力学
例文（典型問題）
「物体が時刻 t₁ に位置 x₁、時刻 t₂ に位置 x₂ にある。平均の速度を求めよ。」
文法（式の立て方）
平均の速度：v̄ = Δx / Δt = (x₂ - x₁) / (t₂ - t₁) 平均の加速度：ā = Δv / Δt = (v₂ - v₁) / (t₂ - t₁)
この文法のポイント
  • Δ（デルタ）は「変化量」。Δx は「変位」であり、道のり（距離）とは違う。
  • 速度はベクトル、速さはスカラー。符号（向き）に注意する。
必要な単語（Vocabulary）
記号用語読み単位意味・注意
x, r位置・変位へんいm向きをもつベクトル。Δx = x₂ - x₁
t時刻・時間じこくs時刻は「いつ」、時間は「どのくらい」
v速度 (velocity)そくどm/sベクトル。速さ (speed) と区別
a加速度 (acceleration)かそくどm/s²速度の変化率
Δ（デルタ）変化量へんかりょう─Δ○ ＝ ○の変化量 ＝ ○₂ − ○₁
G-02 等加速度直線運動の３公式 力学
例文（典型問題）
「初速度 v₀ = 10 m/s で走り出した車が、加速度 a = 2 m/s² で加速した。5秒後の速度と移動距離を求めよ。」
文法（式の立て方）
公式①（v-t 関係）：v = v₀ + at 公式②（x-t 関係）：x = v₀t + ½at² 公式③（v-x 関係）：v² - v₀² = 2ax　← t を消去した形
この文法のポイント
  • ３式のどれを使うか：「何が問われているか」と「何が与えられているか」で決まる。
  • 下向きを正にとれば、重力加速度 a = g = 9.8 m/s²（自由落下・鉛直投射に使う）。
必要な単語（Vocabulary）
記号用語読み単位意味・注意
v₀初速度しょそくどm/st = 0 での速度
v終速度しゅうそくどm/st 秒後の速度
a加速度かそくどm/s²一定（等加速度）であることが条件
x変位へんいm出発点からの正方向の変化量
g重力加速度じゅうりょくかそくどm/s²≈ 9.8 m/s²（問題では 9.8 または 10 を使う）
─自由落下じゆうらっか─v₀ = 0 の鉛直落下。x = ½gt²
─水平投射すいへいとうしゃ─水平：等速直線、鉛直：自由落下
G-03 力のベクトル合成・分解 力学
例文（典型問題）
「θ = 30° の斜面上の物体に働く重力を、斜面に平行な方向と垂直な方向に分解せよ。」
文法（式の立て方）
合力：F = F₁ + F₂　（ベクトル和） 分解（θ が水平となす角のとき）： 水平成分：Fₓ = F cosθ 鉛直成分：Fy = F sinθ 斜面（傾き θ）上での重力分解： 斜面に平行：mg sinθ（下向き） 斜面に垂直：mg cosθ（斜面を押す向き）
この文法のポイント
  • どの方向に θ がついているかで sin と cos が入れ替わる。必ず図を描く。
  • 「つり合い（合力 = 0）」の問題はこの文法の応用。
必要な単語（Vocabulary）
記号用語読み単位意味・注意
F力 (force)ちからNベクトル量
θ角度かくど° or rad力の方向と基準方向のなす角
mg重力じゅうりょくNm：質量、g：重力加速度
N垂直抗力すいちょくこうりょくN面が物体を垂直に押す力
─合力ごうりょくN複数の力をまとめた一つの力
─作用・反作用さよう・はんさよう─第３法則：対の力、大きさ同じ・向き逆
G-04 運動方程式（Newton の第２法則） ★ 最重要 力学
例文（典型問題）
「質量 m = 2 kg の物体が、摩擦係数 μ = 0.3 の水平面上にある。水平方向に力 F = 10 N を加えたとき、加速度を求めよ。」
文法（式の立て方）
運動方程式：ma = F合　（合力 = 質量 × 加速度） 手順： ① 正方向を決める ② 物体にはたらく全ての力を図示（フリーボディダイアグラム） ③ 正方向の合力を ΣF で書く ④ ma = ΣF を立てる
この文法のポイント
  • F は合力（全ての力の和）。一つの力ではない。
  • 正方向を決めることが全ての出発点。符号を間違えると全部崩れる。
  • 摩擦力は運動の向きと逆向きに働く：f = μN
必要な単語（Vocabulary）
記号用語読み単位意味・注意
m質量 (mass)しつりょうkg慣性の大きさ。重さ（重力）とは別概念
a加速度かそくどm/s²ma = F合 の「結果」
ΣF合力ごうりょくN全ての力のベクトル和
f摩擦力まさつりょくNf = μN（動摩擦）、f ≦ μ₀N（静止摩擦）
μ動摩擦係数どうまさつけいすう無次元面の粗さを表す定数
T張力 (tension)ちょうりょくN糸・ロープが引く力
─正方向せいほうこう─自分で決める基準の向き。全問題で最初に設定
G-05仕事（Work）の定義力学
例文（典型問題）
「θ = 60° の方向に 20 N の力を加えて物体を水平に 5 m 動かした。仕事を求めよ。」
文法（式の立て方）
仕事：W = F · d · cosθ （力の大きさ）×（変位の大きさ）×（cosθ：力と変位のなす角のコサイン） 垂直な力がする仕事 = 0（cosθ = cos90° = 0） 仕事率（パワー）：P = W/t = Fv
この文法のポイント
  • cosθ がつく。力と変位が同方向なら W = Fd、垂直なら W = 0。
  • 摩擦力の仕事は負（W < 0）→ エネルギーが失われる。
必要な単語（Vocabulary）
記号用語読み単位意味・注意
W仕事 (work)しごとJスカラー量。符号あり
P仕事率・パワーしごとりつW単位時間あたりの仕事
─保存力ほぞんりょく─重力・弾性力など。する仕事が経路によらない
─非保存力ひほぞんりょく─摩擦力・空気抵抗など。エネルギーを散逸させる
G-06仕事とエネルギーの定理力学
例文（典型問題）
「v₀ = 4 m/s で動いていた質量 2 kg の物体に、合力 6 N を 3 m 加えた。速度を求めよ。」
文法（式の立て方）
仕事・エネルギー定理：W合 = ΔKE = ½mv² - ½mv₀² （全ての力がした仕事の合計）＝（運動エネルギーの変化）
この文法のポイント
  • 運動方程式（G-04）の別バージョン。時間 t を使わずに速度と距離で解ける。
  • W合 には摩擦力の仕事（負）も含める。
必要な単語（Vocabulary）
記号用語読み単位意味・注意
KE運動エネルギーうんどうエネルギーJKE = ½mv²
W合合力のした仕事─J全ての力の仕事の総和
ΔKE運動エネルギーの変化─JKE後 − KE前
G-07力学的エネルギー保存則力学
例文（典型問題）
「高さ h から静かに放した質量 m の物体が、地面に達するときの速度を求めよ（摩擦なし）。」

力学的エネルギー保存則：KE + PE = 一定 ½mv₁² + mgh₁ = ½mv₂² + mgh₂ 【使える条件】保存力のみが仕事をする（摩擦・空気抵抗がない）とき 【使えない条件】摩擦あり → 仕事・エネルギー定理（G-06）を使う

記号用語読み単位意味・注意

G-08運動量と力積力学

G-09運動量保存則力学
例文（典型問題）
「m₁ = 2 kg、v₁ = 3 m/s の物体が、静止した m₂ = 1 kg の物体に衝突し一体になった。衝突後の速度を求めよ。」
文法（式の立て方）
運動量保存則：m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂' 完全非弾性衝突（一体化）：m₁v₁ + m₂v₂ = (m₁ + m₂)V 反発係数（e）：e = −(v₁' - v₂') / (v₁ - v₂) e = 1：弾性衝突（エネルギー保存） e = 0：完全非弾性衝突（一体化）
この文法のポイント
  • 「外力の合計 = 0」のときに成り立つ（孤立系）。
  • 運動量保存則 + 反発係数の式の連立が頻出。
必要な単語（Vocabulary）
記号用語読み単位意味・注意
e反発係数はんぱつけいすう無次元0 ≦ e ≦ 1
─孤立系こりつけい─外力合計 = 0 の系
─弾性衝突だんせいしょうとつ─運動量保存 + 力学的エネルギー保存（e = 1）
─完全非弾性衝突かんぜんひだんせい─一体化する衝突（e = 0）

G-13熱量と比熱熱力学
例文（典型問題）

G-14気体の状態方程式熱力学
例文（典型問題）
「27°C、2 atm の気体（1 mol）の体積を求めよ。」
文法（式の立て方）
状態方程式：PV = nRT P：圧力、V：体積、n：物質量（mol）、R：気体定数、T：絶対温度 変化前後の比較（n 一定）：P₁V₁/T₁ = P₂V₂/T₂
この文法のポイント
  • T は必ず絶対温度（K）：T = t(°C) + 273
  • 「標準状態（0°C, 1 atm）で 22.4 L/mol」も頻出。

G-15熱力学第１法則熱力学
例文（典型問題）
「気体に 500 J の熱を与え、気体が 200 J の仕事をした。内部エネルギーの変化は？」
文法（式の立て方）
ΔU = Q − W ΔU：内部エネルギーの変化 Q：気体が受け取った熱（正 = 吸熱、負 = 放熱） W：気体がした仕事（正 = 膨張） 断熱変化（Q = 0）：ΔU = -W → 膨張すると温度が下がる
必要な単語（Vocabulary）
記号用語読み単位意味・注意
U内部エネルギーないぶエネルギーJ分子の運動エネルギーの合計
W気体がした仕事─JW = PΔV（等圧のとき）
─断熱変化だんねつへんか─Q = 0。外部と熱のやりとりなし

G-19クーロンの法則と電場電磁気
例文（典型問題）
「点電荷 Q から距離 r の点の電場の強さを求めよ。」
文法（式の立て方）
クーロンの法則：F = kq₁q₂/r² 電場の定義：E = F/q₀（単位正電荷にはたらく力） 点電荷の電場：E = kQ/r² 電場中の力：F = qE
必要な単語（Vocabulary）
記号用語読み単位意味・注意
q電荷 (charge)でんかC正・負がある。素電荷 e = 1.6×10⁻¹⁹ C
kクーロン定数─N·m²/C²k ≈ 9.0×10⁹
E電場（電界）でんばN/Cベクトル量。正電荷に働く力の向き
G-20電位とエネルギー電磁気
例文（典型問題）
「点電荷 Q から距離 r の点の電位 V と、そこに電荷 q を置いたときの電気的位置エネルギーを求めよ。」
文法（式の立て方）
点電荷の電位：V = kQ/r 電気的位置エネルギー：U = qV = kqQ/r 電場と電位の関係：E = -ΔV/Δx
必要な単語（Vocabulary）
記号用語読み単位意味・注意
V電位でんいVスカラー量。無限遠を基準（= 0）
U電気的位置エネルギー─JU = qV
─等電位面とうでんいめん─電位が同じ面。電場と垂直
G-21コンデンサー電磁気
例文（典型問題）
「面積 S、間隔 d の平行板コンデンサーに電圧 V をかけた。電荷と蓄えるエネルギーを求めよ。」
文法（式の立て方）
電気容量：C = Q/V　(Q = CV) 平行板：C = ε₀S/d 蓄えるエネルギー：U = ½QV = ½CV² = Q²/(2C) 直列：1/C合 = Σ(1/Cᵢ)　並列：C合 = ΣCᵢ
必要な単語（Vocabulary）
記号用語読み単位意味・注意
C電気容量 (capacitance)でんきようりょうF電荷を蓄える能力
ε₀真空の誘電率ゆうでんりつF/mε₀ ≈ 8.85×10⁻¹² F/m
─誘電体ゆうでんたい─絶縁体。挟むとCが大きくなる
G-22電流・抵抗・オームの法則電磁気
例文（典型問題）
「抵抗 R = 10 Ω に電圧 V = 5 V をかけた。電流と消費電力を求めよ。」
文法（式の立て方）
オームの法則：V = IR 消費電力：P = VI = I²R = V²/R 直列：R合 = ΣRᵢ 並列：1/R合 = Σ(1/Rᵢ)
必要な単語（Vocabulary）
記号用語読み単位意味・注意
I電流 (current)でんりゅうA正電荷の流れる向き（電子と逆）
R抵抗 (resistance)ていこうΩV = IR
P消費電力しょうひでんりょくWP = VI
─起電力 (EMF)きでんりょくV電池が電荷に与えるエネルギー
─内部抵抗ないぶていこうΩ電池の内側の抵抗。端子電圧 = EMF − Ir
G-23キルヒホッフの法則電磁気
例文（典型問題）
「複雑な回路の各抵抗に流れる電流を求めよ。」
文法（式の立て方）
第１法則（節点則）：ΣI_in = ΣI_out　（流入電流 = 流出電流） 第２法則（閉回路則）：Σ（起電力）= Σ（電圧降下）
この文法のポイント
  • 電流の向きを仮定して立式。解が負なら向きが逆。
  • オームの法則だけで解けない複雑回路に使う。
必要な単語（Vocabulary）
記号用語読み単位意味・注意
─節点せってん─回路の分岐点
─閉回路（ループ）へいかいろ─一周できる電流の経路
─電圧降下でんあつこうかV抵抗を通るときの電位の減少（= IR）
G-24磁場とローレンツ力・電磁誘導電磁気
例文（典型問題）
「磁束密度 B の磁場中に速度 v で入った電荷 q の粒子が受ける力を求めよ。」
文法（式の立て方）
ローレンツ力：F = qvB sinθ アンペール力：F = BIL sinθ 電磁誘導（ファラデー）：ε = -dΦ/dt 磁束：Φ = BS cosθ
必要な単語（Vocabulary）
記号用語読み単位意味・注意
B磁束密度じそくみつどT磁場の強さ
Φ磁束じそくWbΦ = BS cosθ
ε誘導起電力ゆうどうきでんりょくV磁束の変化で生じる起電力
─レンツの法則──誘導電流は磁束の変化を打ち消す向きに流れる
─フレミングの左手則──電流・磁場 → 力の向きを決める
G-25交流とインピーダンス電磁気
例文（典型問題）
「実効値 100 V、振動数 50 Hz の交流に R = 10 Ω の抵抗をつないだ。実効電流と消費電力を求めよ。」
文法（式の立て方）
交流電圧：v = V₀ sin(ωt)　（Vrms = V₀/√2：実効値） コイルのリアクタンス：XL = ωL コンデンサーのリアクタンス：XC = 1/(ωC) インピーダンス：Z = √(R² + (XL - XC)²) 共振条件：XL = XC → Z = R（電流最大）
必要な単語（Vocabulary）
記号用語読み単位意味・注意
Vrms実効値じっこうちV直流換算の値。V₀/√2
XL誘導リアクタンスゆうどう─Ωコイルの交流への抵抗
XC容量リアクタンスようりょう─Ωコンデンサーの交流への抵抗
Zインピーダンス─Ω交流回路全体の「抵抗」
─共振きょうしん─XL = XC のとき電流が最大
